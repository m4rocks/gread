GRead

Prefix gread.notangelmario.dev/ or gread.deno.dev/ before the url for a beautiful reader mode

For a cached version of the page, prefix gread.notangelmario.dev/cached/ or gread.deno.dev/cached/ before the url

Example: https://gread.notangelmario.dev/https://notangelmario.dev/blog/young-entrepreneurship
